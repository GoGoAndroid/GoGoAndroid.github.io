Deep Link

Deep link est un bookmarlet qui permet de poser un marqueur dans une page web, afin de permettre de continuer sa lecture lors d'un prochaine visite.

Comment créer un bookmark ?

Sur la page qui vous intéresse, cliquez le bookmarklet, puis à l'endroit que vous souhaitez marquer, cliquer avec la touche contrôle enfoncée.

Comment revenir sur le bookmark ultérieurement ?

Sur la page que vous avez marquée, cliquez sur le bookmarklet, si une marque a été posée, une message vous demandera si vous souhaitez revenir à l'endroit marqué.
